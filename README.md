> DeepLearning Log Send to DB Module  

## Introdution
Log2DB is a tool that uploads, manages, and visualizes simple deep learning logs to your own DB.  

## How To Install 
- Download manually:  
https://github.com/Oldentomato/Log2DB/releases  

Download PyPI:
```sh
pip install log2db
```
